# Brand Variants

Each subfolder is a self-contained icon set with full PNG exports.

| Folder | Style | Best for |
|---|---|---|
| `neural-gradient/` | Gradient brain + neural network + orange connectors | Primary — matches main brand assets |
| `flat-solid/` | Flat solid #0056b3 blue — no gradients | Monochrome print, embossing, embroidery |
| `geometric/` | Wireframe diamond + cyan glow + orange ports | Dark backgrounds, tech/dev contexts, t-shirts |
| `badge/` | Dark navy circle + grid + connector shield | Stickers, profile badges, Discord server icon |

## Each folder contains

| File | Description |
|---|---|
| `icon.svg` | Source vector — transparent/native bg |
| `lockup.svg` | Horizontal logo (icon + wordmark) |
| `icon-64.png` | 64×64 |
| `icon-128.png` | 128×128 |
| `icon-192.png` | 192×192 |
| `icon-256.png` | 256×256 |
| `icon-512.png` | 512×512 |
| `icon-1024.png` | 1024×1024 |
| `android-chrome-192.png` | Android / PWA |
| `android-chrome-512.png` | Android splash |
| `apple-touch-icon.png` | 180×180 iOS |
| `lockup-1200x400.png` | OG / LinkedIn |
| `banner-1500x500.png` | Twitter/X banner |
